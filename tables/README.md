# Tables

Generated result tables are not tracked in this repository by default.

The reproduction scripts generate CSV tables for:

- synthetic sample-level metrics;
- synthetic event-level metrics;
- baseline comparisons;
- FORESEE field-case summaries;
- DAS-N2N versus MCS-AND field comparisons;
- sensitivity and robustness analysis.

For publication or review, final CSV tables can be archived on Zenodo together with the released code.
